# Document Sync — Background Processing Demo

Oracle APEX application export demonstrating background processing using Execution Chain + Monitor.

## What this demo shows

This application illustrates how to run a long task in the background using an Execution Chain, while keeping the UI responsive and displaying live progress on a Monitor page.

## Files

- `f122.sql` — Oracle APEX application export

- ## How to use

- 1. Open Oracle APEX App Builder
  2. 2. Import the application from `f122.sql`
     3. 3. Ensure the parsing schema contains the required database objects
        4. 4. Run the application and execute the document sync flow
